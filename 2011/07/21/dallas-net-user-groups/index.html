Dallas .NET User Groups

By randyburden on July 21, 2011

Here is a handy list of Dallas .NET user groups. They all typically meet once a month but sometimes it changes.

  • Dallas .NET User Group
    • 2nd Thursday 6 – 9pm
  • North Dallas .NET User Group
    • 1st Wednesday 6 – 8:30pm

  • Dallas ASP.NET User Group
    • 4th Tuesday 6 – 9pm
    • $25/annual membership fee or you can pay as you go at $1.61/per session
  • Hands-On-Coding.net
    • 2nd Tuesday 6 – 9pm
  • Dallas CSharp SIG
    • 1st Thursday 6:30 – 8:30pm
  • D/FW Connected Systems User Group
    • 2nd Wednesday 6 – 8:30pm
You can find a more complete list on the INETA website: INETA.org – Browse User Groups
and more on Microsoft’s Technical Community website: Microsoft | UGSS
